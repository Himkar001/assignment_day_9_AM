# Day -9
# Part- A

This repository contains solutions for the Day 9 Python assignment covering advanced list operations.

## Files

student_system.py  
Implements a Student Management System using lists.

Features:
- Add student
- Show toppers
- Calculate class average
- Show above average students
- Remove student
- Save records to file

students.txt  
Stores student records generated by the program.

matrix_ops.py  
Implements matrix operations using Python lists.

Functions:
- Matrix Addition
- Matrix Transpose
- Matrix Multiplication

part_A.txt  
Explanation of the Student Management System logic.

ai_pair_sum.md  
Analysis of AI generated code for pair sum problem and improved implementation.

## Concepts Used

- Lists
- List Comprehension
- Sorting with lambda
- File Handling
- Nested Lists
- Matrix Operations
