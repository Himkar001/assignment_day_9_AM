PART B — Matrix Operations

This file contains the output results of the matrix operations program.

Matrix A
[1, 2]
[3, 4]

Matrix B
[5, 6]
[7, 8]

Matrix Addition Result
[[6, 8], [10, 12]]

Matrix Transpose Result
[[1, 3], [2, 4]]

Matrix Multiplication Result
[[19, 22], [43, 50]]

Explanation

1. Matrix Addition
Each element from matrix A is added to the corresponding element in matrix B.

Example:
1 + 5 = 6
2 + 6 = 8

2. Matrix Transpose
Rows are converted into columns.

Example:
[1,2]
[3,4]

becomes

[1,3]
[2,4]

3. Matrix Multiplication
Each element is computed using the dot product of rows and columns.

Example:
(1×5 + 2×7) = 19
(1×6 + 2×8) = 22
